# Pluie Texture Pack

A gentle, rain-inspired texture pack for Minecraft.

## Current Version

**Beta 1.7.3** - Complete texture pack with soft, clean aesthetics.

## Quick Start

```bash
npm run build:b1.7.3
```

Outputs to `output/Pluie-Texture-Pack-b1.7.3.zip`

## Texture Editing

```bash
npm run aseprite b1.7.3/terrain.png
```

Requires `ASEPRITE_PATH` environment variable.

## Project Structure

- `b1.7.3/` - Beta 1.7.3 textures
- `scripts/` - Build and development tools
- `output/` - Generated texture packs
